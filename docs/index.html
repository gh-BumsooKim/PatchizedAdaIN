Spatial Color-Aware Arbitrary Style Transfer using patchization-based Adaptive Instance Normalization with Latent Code Blending

Bumsoo Kim1, Sanghun Seo*
1,*School of Art and Technology, Chung-Ang University, Korea
*Indicates Corresponding Author
Paper (Soon) Code

We proposed PatchizedAdaIN operation for color-aware arbitrary style transfer. It can preserve regional color distribution without any additional computational overhead.

Abstract

Neural style transfer or arbitrary style transfer has a objective that perceptually change the image style by referencing style image information for artistic cre- ations with visual aesthetics. However, existing style transfer network underlying the AdaIN operation can not capture the spatially local color distribution due to the limitation that AdaIN align the the image style using global statistics. Thus, outputted image has highly similar appearance regardless of geometric trans- formation of inputted images. To address it, we propose the color-aware style transfer system that can reflect the spatial color distribution of reference image by using patchized AdaIN(PatchizedAdaIN). Our key idea is to adopt the AdaIN layer at each divided patch with corresponding partial statistics, and reconstruct these patches in latent space. By doing that, our style transfer can perform color- aware style transfer according to the geometric transformation with local color statistics. Since PatchizedAdaIN operation underlying the existing AdaIN, our system can readily transfer the style of content image both local and global geo- metric features without additional learnable parameters. Also, we provide code blending control parameter to manipulate the quality of output for plausible appearance. It allows the user to freely output the different result by chang- ing the code blending level. From experimental results, we demonstrate that our method can reflect the geometric transformation (e.g., translation, rotation, flip) of style image with our designed PatchizedAdaIN. In addition, we also show sim- ple extension scheme for existing networks to enable spatial color-aware arbitrary style transfer by replacing AdaIN to PatchizedAdaIN.
